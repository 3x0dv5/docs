.. _products:

***************************************************
 MapServer Suite Products
***************************************************

.. note::
    Depuis la version 6.0 de MapServer, le projet MapServer inclus une suite de produits Open Source 
    afin de fournir un ensemble complet d'outils de cartographie web à la communauté. Le Comité de 
    Gestion de Projet de MapServer maintient tout ces produits au sein de  MapServer.
    
MapServer
----------

Le code de source de MapServer, écrit en C, et toujours l'un des plus rapide et plus configurable 
des moteurs cartographiques web au monde.

- :ref:`Documentation <documentation>` 
- :ref:`Téléchargement <source>`
- `Github <https://github.com/mapserver/mapserver/>`__

MapCache
--------

À partir de MapServer 6.0, MapServer inclus un outil puissant de cache de tuiles via le projet MapCache.

- :ref:`Documentation <mapcache>` 
- :ref:`Téléchargement <source>`
- `Github <https://github.com/mapserver/mapcache/>`__

TinyOWS
--------

À partir  de MapServer 6.0, MapServer inclus également la possibilité fortement demandée de réaliser des 
requêtes transactionnelles (édition en ligne de géométries) via la spécification WFS, en utilisant le 
projet TinyOWS.

- :ref:`Documentation <tinyows>` 
- :ref:`Téléchargement <source>`
- `Github <https://github.com/mapserver/tinyows/>`__

.. note::
	Les Développeurs désirant contribuer aux projets Github doivent lire le 
	:ref:`guide pour Github <git>` de MapServer.
